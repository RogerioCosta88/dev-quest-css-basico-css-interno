CSS Interno

No CSS Interno os estilos também são criados no HTML, mas diferentemente do CSS inline que estiliza cada Tag separadamente, o CSS Interno consegue estilizar todos os paragráfos de uma só vez.

Também é utilizado uma Tag style, mas aqui ela é colocada no "corpo"da Tag head.

Esta forma de estilizar também não é muito boa, pois, se temos mais de uma página no site, teremos que colocar uma Tag style dentro da Tag head em cada página. Se o site conter muitas páginas, isso custará muito tempo.
